Использование
Запустите программу.
Введите данные в формате: Фамилия Имя Отчество датарождения номертелефона пол,
разделенные пробелами.
Программа проверит данные и, если они введены верно,
создаст файл с фамилией пользователя и запишет туда введенные данные.
Если данные введены неверно, программа выведет сообщение об ошибке.
Примеры использования
Ввод
Copy code
Иванов Иван Иванович 01.01.1990 1234567890 m
Вывод
Copy code
Данные успешно сохранены.
Содержимое файла Иванов.txt
Copy code
Иванов Иван Иванович 01.01.1990 1234567890 m
Примечания
Дата рождения должна быть в формате dd.MM.yyyy.
Номер телефона должен быть целым положительным числом.
Пол должен быть указан как f или m.
